Neo4j Browser (Stealth Release)

Neo4j Browser is a stealth project to update Neo4j's built-in client application. Please do not share links to this project or discuss it in any public forums.

  • developer focused
  • uses only public APIs
  • for modern browsers, with reasonable fallbacks
